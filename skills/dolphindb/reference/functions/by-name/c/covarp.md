# covarp

首发版本：3.00.5

## 语法

`covarp(X, Y)`

## 详情

计算 *X* 和 *Y* 的总体协方差。

## 参数

**Y** 和 **X** 是标量、相同长度的数值型向量、维度相同的矩阵或表。若 *X* 是表，只对其内数值列进行计算。

## 返回值

* 当 *X* 和 *Y* 均为标量或等长向量时，返回 DOUBLE 类型标量。
* 当 *X* 或 *Y* 为矩阵时，返回 DOUBLE 类型向量。
* 当*X* 或 *Y* 为表时，返回一个表。

## 例子

```
x = [100, 100, 100, 1100]
y = [0.98, 0.95, 0.92, 0.88]
covarp(x,y);

// output: -13.13

x=matrix([0.010,  0.012, -0.005, 0.008], [0.009,  0.011, -0.004, 0.007], [0.011,  0.013, -0.006, 0.009])
y=[0.008, 0.010, -0.003, 0.006]
covarp(x,y)

// output: 3
```
