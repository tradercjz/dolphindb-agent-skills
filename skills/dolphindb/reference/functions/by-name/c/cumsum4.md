# cumsum4

## 语法

`cumsum4(X)`

参数说明和窗口计算规则请参考：累计窗口系列（cum 系列）

## 详情

计算 *X* 元素的累计四次方和。

## 返回值

DOUBLE 类型，其数据形式同 *X*。

## 例子

```
x=[2,3,4];
cumsum4 x;
// output
[16,97,353]

m=matrix(1 2 3, 4 5 6);
m;
```

| #0 | #1 |
| --- | --- |
| 1 | 4 |
| 2 | 5 |
| 3 | 6 |

```
cumsum4(m);
```

| #0 | #1 |
| --- | --- |
| 1 | 256 |
| 17 | 881 |
| 98 | 2177 |

相关函数：sum4
