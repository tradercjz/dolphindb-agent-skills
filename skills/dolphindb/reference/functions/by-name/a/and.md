# and

## 语法

`and(X, Y)` 或 `X && Y`

## 详情

对 *X* 和 *Y* 内的元素逐个进行逻辑与（logical AND）的运算并返回结果。

## 参数

**X** 和 **Y** 可以是标量、数据对、向量或矩阵。

## 返回值

数据类型为布尔类型。

若**X** 和 **Y**至少一个是矩阵，则返回矩阵；否则，若至少一个是数据对，则返回数据对;否则，若至少有一个是向量，则返回向量;若**X** 和 **Y**均为标量，则返回标量。

## 例子

```
1 && 3;
// output
1

x=1 2 3
x && 0
// output
[0,0,0]

x=1 2 3
y=0 1 0
x && y
// output
[0,1,0]

t=table(1 2 2 3 as id, 4 5 6 5 as value)
t
```

| id | value |
| --- | --- |
| 1 | 4 |
| 2 | 5 |
| 2 | 6 |
| 3 | 5 |

```
select id, value from t where id=2 and value=5;   // SQL query
```

| id | value |
| --- | --- |
| 2 | 5 |
