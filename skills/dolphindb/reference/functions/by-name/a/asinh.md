# asinh

## 语法

`asinh(X)`

## 详情

计算 *X* 的反双曲正弦。

## 参数

**X** 可以是标量、向量、矩阵或表。

## 返回值

返回结果的数据形式与输入保持一致：若输入为标量，返回标量；若输入为向量、矩阵或表，返回相同维度的结果。

## 例子

```
asinh(0.841471 0.909297 0.141120);
// output
[0.764725,0.815761,0.140656]
```

相关函数：asin, acos, atan, sin, cos, tan, acosh, atanh, sinh, cosh, tanh
