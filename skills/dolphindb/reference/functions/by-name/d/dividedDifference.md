# dividedDifference

## 语法

`dividedDifference(X, Y, resampleRule, [closed='left'], [origin='start_day'],
[outputX=false])`

## 详情

该函数根据 *resampleRule*, *closed*, *origin* 确定的采样规则，对
*X* 进行重采样操作。并根据重采样后的 *X*，对 *Y* 进行均差插值（DividedDifference
Interpolation）。

若不指定 *outputX*，仅返回一个对 *Y* 插值后的向量。若指定 *outputX* =
true，则返回一个 tuple，其第一个元素为 *X* 重采样后的向量，第二个元素为对 *Y* 插值后的向量。

## 参数

**X** 严格递增的时间类型向量。

**Y** 同 *X* 等长的数值型向量。

**resampleRule** 一个字符串，可选值请参考 resample 的
*rule* 参数。

**closed** 和 **origin** 同 resample 的
*closed* 和 *origin* 参数。

**outputX** 布尔类型，表示是否输出 *X* 按照 *resampleRule*, *closed*,
*origin* 重采样后的向量。默认值为 false。

## 例子

**例
1**

```
dividedDifference([2016.02.14 00:00:00, 2016.02.15 00:00:00, 2016.02.16 00:00:00], [1.0, 2.0, 4.0], resampleRule=`60min);

// output
[1,1.0217,1.0451,1.0703,1.0972,1.1259,1.1562,1.1884,1.2222,1.2578,
1.2951,1.3342,1.375,1.4175,1.4618,1.5078,1.5556,1.605,1.6562,1.7092,
1.7639,1.8203,1.8785,1.9384,2,2.0634,2.1285,2.1953,2.2639,2.3342,
2.4062,2.48,2.5556,2.6328,2.7118,2.7925,2.875,2.9592,3.0451,3.1328,
3.2222,3.3134,3.4062,3.5009,3.5972,3.6953,3.7951,3.8967,4]
```

**例 2** 不同 *closed*
取值的影响。

```
X = 2022.01.01T00:00:00 + [0, 3, 6, 9] * 60  // 00:00, 00:03, 00:06, 00:09
Y = [1.0, 3.0, 7.0, 13.0]

// closed='left'（默认）：区间为 [t, t+3min)，左闭右开
// 00:03 归入 [00:03, 00:06)，是第二个桶的起点
dividedDifference(X, Y, `3min, closed=`left)

// closed='right'：区间为 (t, t+3min]，左开右闭
// 00:03 归入 (00:00, 00:03]，成为第一个桶的终点
// 重采样 X 的划分基准不同，插值结果随之变化
dividedDifference(X, Y, `3min, closed=`right)
```

**例 3** 不同 *origin* 取值的影响。

```
// 数据从 00:00:30 开始，不与整分对齐
X = 2022.01.01T00:00:30 + (0..4) * 60  // 00:00:30, 00:01:30, ..., 00:04:30
Y = [2.0, 4.0, 7.0, 11.0, 16.0]

// origin='start_day'（默认）：从当天零点 00:00:00 对齐
// 时间桶: [00:00:00, 00:01:00), [00:01:00, 00:02:00), ...
// 重采样 X 为: 00:00:00, 00:01:00, 00:02:00, 00:03:00, 00:04:00
dividedDifference(X, Y, `1min, origin=`start_day, outputX=true)

// origin='start'：从第一个数据点 00:00:30 对齐
// 时间桶: [00:00:30, 00:01:30), [00:01:30, 00:02:30), ...
// 重采样 X 为: 00:00:30, 00:01:30, 00:02:30, 00:03:30, 00:04:30
dividedDifference(X, Y, `1min, origin=`start, outputX=true)

// origin=自定义时间戳：从 00:00:10 对齐
// 时间桶: [00:00:10, 00:01:10), [00:01:10, 00:02:10), ...
// 重采样 X 为: 00:00:10, 00:01:10, 00:02:10, 00:03:10, 00:04:10
dividedDifference(X, Y, `1min, origin=2022.01.01T00:00:10, outputX=true)
```

**例 4** 不同 *outputX*
取值的影响。

```
X = [2016.02.14T00:00:00, 2016.02.15T00:00:00, 2016.02.16T00:00:00]
Y = [1.0, 2.0, 4.0]

// outputX=false（默认）：只返回插值后的 Y 向量
dividedDifference(X, Y, `60min)
// 输出: [1, 1.0217, ..., 4]（49 个点）

// outputX=true：返回 tuple，[0] 为重采样后的 X，[1] 为插值后的 Y
result = dividedDifference(X, Y, `60min, outputX=true)
result[0]  // 每小时的时间戳，共 49 个
result[1]  // 对应的插值 Y 值，共 49 个
```
