# 金融分析

## 金融分析

- [`arima`](../by-name/a/arima.md) — 差分整合移动平均自回归（ARIMA, Autoregressive Integrated Moving Average）模型是一种单变量时间序列分析模型，主要由三部分构成，分别为自回归模型（AR）、差分过程（I）和移动平均模型（MA）。
- [`nss`](../by-name/n/nss.md) — 使用 NSS 模型和指定的优化方法，拟合债券的收益率曲线。
- [`ns`](../by-name/n/ns.md) — 使用 NS 模型和指定的优化方法，拟合债券的收益率曲线。
- [`condValueAtRisk`](../by-name/c/cvar.md) — 条件风险指标（Conditional Value at Risk；也可称 Expected Shortfall），用于计算超过风控指标的损失的平均值。该函数将返回其绝对值，为 DOUBLE 类型。如果没有收益率低于 VaR 的值，则返回 VaR 的值。
- [`nssPredict`](../by-name/n/nsspredict.md) — 根据指定的 NS 模型或 NSS 模型，预测债券的收益率。
- [`trueRange`](../by-name/t/trueRange.md) — 计算真实波幅，即 | 最高价-最低价 | 和 | 最高价-昨收 | 和 | 昨收-最低价 | 的最大值。
- [`valueAtRisk`](../by-name/v/var_0.md) — 风控指标（Value at Risk），用于量化在给定的置信水平（例如 95% 或 99%）和特定时间范围内金融资产可能遭受的最大损失。该函数将返回最低收益率的绝对值，为 DOUBLE 类型。
- [`varma`](../by-name/v/varma.md) — 使用向量自回归移动平均模型（Vector Autoregressive Moving-Average Model，简称 VARMA 模型）来分析多变量时间序列。返回一个字典，表示 VARMA 模型的分析结果，详细说明请参见“返回值”小节。
- [`bondCashflow`](../by-name/b/bondCashflow.md) — 计算面值为 100 元的债券的现金流，支持固息债券，零息债券和贴现债券。
- [`bondYield`](../by-name/b/bondyield.md) — 通过债券净价(Clean Price)或全价(Dirty Price)计算债券的到期收益率 (Yield To Maturity)。
- [`treasuryConversionFactor`](../by-name/t/treasuryconversionfactor.md) — 本函数基于[中金所国债转换因子和应计利息计算公式](http://www.cffex.com.cn/5tf/)对转换因子进行估值计算。成功执行后将返回转换因子的估值。
- [`vanillaOption`](../by-name/v/vanillaoption.md) — 对香草期权（Vanilla Option）进行估值计算。
- [`maxDrawdown`](../by-name/m/maxdrawdown.md) — 计算传入向量数据的最大回撤（Maximum Drawdown，简称 MDD）或最大回撤率（Maximum Drawdown Rate）。
- [`mdd`](../by-name/m/mdd.md)
- [`cummdd`](../by-name/c/cummdd.md) — 累计计算传入向量数据的最大回撤。

## FICC

- [`bondAccrInt`](../by-name/b/bondaccrint.md) — 计算债券的应计利息（Accrued Interest）。应计利息是指债券自上一次付息日至交易结算日之间累积的未付利息。
- [`bondCalculator`](../by-name/b/bondCalculator.md) — 该函数用于实现债券到期收益率、净价和全价三者之间的互算，同时支持计算久期、凸度等风险指标。
- [`bondInstrumentCalculator`](../by-name/b/bondinstrumentcalculator.md) — 债券计算器，实现债券价格 (ytm / cleanPrice / dirtyPrice) 三者互算，同时计算久期、凸度等风险指标。
- [`bondConvexity`](../by-name/b/bondconvexity.md) — 返回面值为 100 的债券的凸性。凸性指债券价格与利率间非线性关系的一种量度，表示为债券价格对利率的二阶导数。
- [`bondDirtyPrice`](../by-name/b/bondDirtyPrice.md) — 返回面值为 100 的债券的全价。
- [`bondDuration`](../by-name/b/bondDuration.md) — 返回面值为 100 的债券的麦考利久期（Macauley Duration）。麦考利久期利用折现后的债券现金流的加权平均来计算债券的到期时间。
- [`bondFuturesPricer`](../by-name/b/bondFuturesPricer.md) — 计算国债期货合约的净现值。
- [`bondPricer`](../by-name/b/bondPricer.md) — 对债券进行定价，输出净现值（NPV），同时可计算多种风险指标，包括一阶敏感度（Delta）、二阶敏感度（Gamma）以及关键利率久期（Key Rate Duration）。
- [`bondYieldCurveBuilder`](../by-name/b/bondYieldCurveBuilder.md) — 构建债券收益率曲线，可作为定价时使用的贴现曲线（即期收益率曲线）。
- [`cmFutAmericanOptionPricer`](../by-name/c/cmfutamericanoptionpricer.md) — 对商品期货美式期权进行定价。
- [`cmFutEuropeanOptionPricer`](../by-name/c/cmfuteuropeanoptionpricer.md) — 对商品期货欧式期权进行定价。
- [`cmFutVolatilitySurfaceBuilder`](../by-name/c/cmFutVolatilitySurfaceBuilder.md) — 构建商品期货期权波动率曲面。
- [`curvePredict`](../by-name/c/curvePredict.md) — 在给定的曲线上，预测指定点位（由 *dt* 确定）的曲线值。目前仅支持零息利率曲线。
- [`eqAmericanOptionPricer`](../by-name/e/eqamericanoptionpricer.md) — 对权益类美式期权进行定价。
- [`eqEuropeanOptionPricer`](../by-name/e/eqeuropeanoptionpricer.md) — 对权益类欧式期权进行定价。
- [`eqProxyVolatilitySurfaceBuilder`](../by-name/e/eqproxyvolatilitysurfacebuilder.md) — 构造权益类代理波动率曲面。
- [`eqVolatilitySurfaceBuilder`](../by-name/e/eqvolatilitysurfacebuilder.md) — 构建权益类期权波动率曲面。
- [`fxEuropeanOptionPricer`](../by-name/f/fxEuropeanOptionPricer.md) — 使用 BlackScholes Model 和 Analytic Method，对外汇欧式期权进行定价，返回期权的净现值（NPV）。
- [`fxForwardPricer`](../by-name/f/fxForwardPricer.md) — 基于给定的即期汇率、本币与外币的贴现曲线，以及定价日期，计算外汇远期合约的净现值（NPV）。
- [`fxSwapPricer`](../by-name/f/fxSwapPricer.md) — 计算外汇掉期的净现值。
- [`fxVolatilitySurfaceBuilder`](../by-name/f/fxVolatilitySurfaceBuilder.md) — 构建外汇波动率曲面。
- [`instrumentPricer`](../by-name/i/instrumentPricer.md) — 对一个或多个金融合约（可为相同类型或不同类型）进行批量定价。
- [`irCrossCurrencyCurveBuilder`](../by-name/i/irCrossCurrencyCurveBuilder.md) — 构建跨币种利率互换收益率曲线。
- [`irDepositPricer`](../by-name/i/irDepositPricer.md) — 计算存款（Certificate of Deposit, CD）的定价。
- [`irFixedFloatingSwapPricer`](../by-name/i/irFixedFloatingSwapPricer.md) — 计算固定利率互换（IR Fixied-Floating Swap, IRS）的定价。
- [`irSingleCurrencyCurveBuilder`](../by-name/i/irSingleCurrencyCurveBuilder.md) — 构建单货币利率互换收益率曲线。目前仅支持 CNY\_FR\_007 和 CNY\_SHIBOR\_3M 两条曲线构建。
- [`optionVolPredict`](../by-name/o/optionVolPredict.md) — 给定点位的时间和行权价，从指定的波动率曲面上预测点位对应的波动率。
- [`portfolioPricer`](../by-name/p/portfolioPricer.md) — 对一个或多个金融合约（可为相同类型或不同类型）进行组合定价。

## 实时计算引擎及工具函数

- [`createMktDataEngine`](../by-name/c/createMktDataEngine.md) — 创建市场数据（曲线/曲面等）实时构建引擎。
- [`createPricingEngine`](../by-name/c/createpricingengine.md) — 创建实时估值定价引擎。该引擎支持调用 DolphinDB 内置估值定价函数，也支持用户传入自定义函数。
- [`appendMktData`](../by-name/a/appendMktData.md) — 向市场数据构建引擎或者定价引擎中写入市场数据。
