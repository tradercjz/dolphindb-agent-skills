# 机器学习

## 机器学习

- [`adaBoostClassifier`](../by-name/a/adaBoostClassifier.md) — 进行 AdaBoost 分类。
- [`adaBoostRegressor`](../by-name/a/adaBoostRegressor.md) — 进行 AdaBoost 回归。
- [`beta`](../by-name/b/beta.md) — 返回 *Y* 在 *X* 上的回归系数的最小二乘估计。
- [`bvls`](../by-name/b/bvls.md) — 执行有界变量最小二乘法（Bounded-Variable Least-Square）。
- [`elasticNet`](../by-name/e/elasticNet.md) — 进行弹性网络回归（ElasticNet 回归）计算。即最小化以下目标函数：
- [`elasticNetCV`](../by-name/e/elasticNetCV.md) — 使用五折交叉验证方法进行弹性网络回归（ElasticNet），输出最优参数对应的模型。
- [`gaussianNB`](../by-name/g/gaussianNB.md) — 使用高斯朴素贝叶斯（Gaussian Naive Bayes）算法对数据进行分类训练。
- [`glm`](../by-name/g/glm.md) — 训练广义线性模型。
- [`gmm`](../by-name/g/gmm.md) — 通过数据集训练 GMM（高斯混合模型）。
- [`kernelRidge`](../by-name/k/kernelRidge.md) — 该函数基于核函数和岭回归的正则化技术，拟合数据间的复杂非线性关系，输出回归预测模型。
- [`kmeans`](../by-name/k/kmeans.md) — 对训练集执行 K-Means 聚类。
- [`knn`](../by-name/k/knn.md) — 通过 K 邻近算法（暴力搜索法）对表中的数据进行训练。
- [`lasso`](../by-name/l/lasso.md) — 进行 lasso 回归估计。
- [`lassoBasic`](../by-name/l/lassoBasic.md) — 进行 lasso 回归估计。最小化以下目标函数：
- [`lassoCV`](../by-name/l/lassoCV.md) — 使用五折交叉验证方法进行 lasso 回归估计，输出最优参数对应的模型。
- [`logisticRegression`](../by-name/l/logisticRegression.md) — 对数据源中的自变量（*xColNames*）和因变量（*yColName*）进行逻辑回归分析。生成的模型可以作为 `predict` 函数的输入，用于预测新数据的分类结果。
- [`mmse`](../by-name/m/mmse.md) — 在给定长度（以元素个数衡量）的滑动窗口内，将因变量 *Y* 与自变量 *X* 进行普通最小二乘回归的系数估计并计算均方误差。计算公式如下：
- [`msl`](../by-name/m/mslr.md) — 在长度为 *window* 的滑动窗口中，将因变量 *Y* 与自变量 *X* 进行普通最小二乘回归。返回一个元组，其中第一个元素为截距，第二个元素为斜率。
- [`multinomialNB`](../by-name/m/multinomialNB.md) — 使用多项式朴素贝叶斯（Multinomial Naive Bayes）算法对数据进行分类训练。
- [`ols`](../by-name/o/ols.md) — 返回对 *X* 和 *Y* 计算普通最小二乘回归的结果。需要注意的是，该函数会将 *X* 和 *Y* 中的空值替换为0后进行计算。
- [`olsEx`](../by-name/o/olsEx.md) — 返回对 *X* 和 *Y* 计算普通最小二乘回归的结果。*X* 和 *Y* 是分布式表中的列。
- [`piecewiseLinFit`](../by-name/p/piecewiselinfit.md) — 通过给定数据点集和给定线段数，拟合一个连续的分段线性回归函数；并使用差分进化算法通过最小化平方误差之和，找到给定数量的线段断点的最优位置。
- [`poly1d`](../by-name/p/poly1d.md)
- [`polyPredict`](../by-name/p/polyPredict.md) — 利用给定的多项式系数 *z* 和自变量 *x*，计算因变量的值。
- [`polyFit`](../by-name/p/polyfit.md) — 使用最小二乘法基于给定的数据点 *X* 和 *Y*，计算 n 次多项式的最佳拟合系数。
- [`polynomial`](../by-name/p/polynomial.md) — 对于每个 *X* 中的元素，用多项式系数计算一个值。
- [`predict`](../by-name/p/predict.md) — 用特定的模型对数据进行预测。
- [`pwlfPredict`](../by-name/p/pwlfpredict.md) — 本函数须结合函数 [piecewiseLinFit](piecewiselinfit.html) 共同使用，即使用 `piecewiseLinFit` 拟合后的连续分段线性回归模型来对输入的数据点进行预测。
- [`randomForestClassifier`](../by-name/r/randomForestClassifier.md) — 计算指定数据源中 *yColName* 和 *xColNames* 随机森林分类。
- [`randomForestRegressor`](../by-name/r/randomForestRegressor.md) — 计算指定数据源中 *yColName* 和 *xColNames* 随机森林回归。
- [`residual`](../by-name/r/residual.md) — 返回对 *X* 和 *Y* 计算最小二乘回归结果的残差。
- [`ridge`](../by-name/r/ridge.md) — 进行 ridge 回归估计。
- [`ridgeBasic`](../by-name/r/ridgeBasic.md) — 进行 ridge 回归估计。
- [`vectorAR`](../by-name/v/vectorar.md) — 使用向量自回归模型（Vector Autoregression model，简称 VAR 模型）来分析多变量时间序列。
- [`wls`](../by-name/w/wls.md) — 返回对 *X* 和 *Y* 计算加权最小二乘回归的结果。
- [`garch`](../by-name/g/garch.md) — 使用广义自回归条件异方差模型（Generalized Autoregressive Conditional Heteroskedasticity，简称 GARCH 模型）来分析单变量时间序列。返回一个字典，表示 GARCH 模型的分析结果，详细说明请参见“返回值”小节。

## 文本处理

- [`loadVocab`](../by-name/l/loadVocab.md) — 将指定路径下的词库加载到内存中，用于文本向量化处理。
- [`tokenizeBert`](../by-name/t/tokenizeBert.md) — 应用指定词库对输入文本 *text* 进行分词。本函数使用 WordPiece 分词算法，适用于 BERT（Bidirectional Encoder Representations from Transformers） 模型。
- [`unloadVocab`](../by-name/u/unloadVocab.md) — 从内存中删除已加载的词库。
