# 运算符

## 逻辑

- [`and`](../by-name/a/and.md) — 对 *X* 和 *Y* 内的元素逐个进行逻辑与（logical AND）的运算并返回结果。
- [`bitAnd`](../by-name/b/bitAnd.md) — 返回位运算与（`bitAnd`）的结果。
- [`bitOr`](../by-name/b/bitOr.md) — 返回位运算或（`bitOr`）的结果。
- [`bitXor`](../by-name/b/bitXor.md) — 返回位运算异或（`bitXOr`）的结果。
- [`not`](../by-name/n/not.md) — 返回对 *X* 应用 NOT 的值。返回值是 0, 1 或 NULL。 0 的 NOT 值是 1；NULL 的 NOT 值仍是 NULL；所有其他值的 NOT 值是 0。
- [`or`](../by-name/o/or.md) — 按元素逐个返回 *X* 逻辑或 `(OR)Y` 的结果。
- [`xor`](../by-name/x/xor.md) — 按元素逐个返回 *X* 逻辑异或 (`XOR`)*Y* 的结果。

## 关系

- [`between`](../by-name/b/between.md) — 检查 *X* 的每个元素是否在 *Y* 确定的范围内（两个边界都是包含在内的）。返回结果的长度或维度与 *X* 相同。
- [`eq`](../by-name/e/eq.md) — 如果 *X* 和 *Y* 都不是集合，返回逐个元素比较 *X*==*Y* 的结果。
- [`eqFloat`](../by-name/e/eqFloat.md) — 以给定（小数点后位数）精度，逐个元素比较 *X*==*Y*。
- [`eqObj`](../by-name/e/eqObj.md) — 检验两个对象的类型和值是否相同。只有当类型和值都相同时，此函数才会返回 true。 如果值相同但类型不同，则此函数仍返回 false，这与函数 [eq](eq.html) 不同。
- [`eqPercent`](../by-name/e/eqpercent.md) — 根据自定义约束条件，判断两个输入参数的每个对应位置上的元素是否都相等。
- [`ge`](../by-name/g/ge.md) — 如果 *X* 和 *Y* 都不是集合，返回逐个元素比较 *X*>=*Y* 的结果。
- [`gt`](../by-name/g/gt.md) — 如果 *X* 和 *Y* 都不是集合，返回逐个元素比较 *X*>*Y* 的结果。

- [`le`](../by-name/l/le.md) — 如果 *X* 和 *Y* 都不是集合，返回逐个元素比较 *X*<=*Y* 的结果。
- [`lt`](../by-name/l/lt.md)
- [`ne`](../by-name/n/ne.md) — 如果 *X* 和 *Y* 都不是集合，逐个元素地比较 *X* 和 *Y*，如果不相等则返回1。
