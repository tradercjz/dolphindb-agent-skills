- [`tan`](../by-name/t/tan.md) — 返回 *X* 的正切。
- [`tanh`](../by-name/t/tanh.md) — 返回 *X* 的双曲正切。
- [`tril`](../by-name/t/tril.md) — * 若未指定 *k*： 返回矩阵 *X* 的下三角部分，其余元素设为0。 * 若指定 *k*： 返回矩阵 *X* 的第 *k* 条对角线上以及该对角线下方的元素，其余元素设为0。矩阵的主对角线为其第0条对角线。
- [`triu`](../by-name/t/triu.md) — * 若未指定 *k*： 返回矩阵 *X* 的上三角部分，其余元素设为0。 * 若指定 *k*： 返回矩阵 *X* 的第 *k* 条对角线上以及该对角线上方的元素，其余元素设为0。矩阵的主对角线为其第0条对角线。

- [`signbit`](../by-name/s/signbit.md) — 获取输入数据的符号位。

## 相关性

- [`acf`](../by-name/a/acf.md) — 计算 *X* 的1阶至 *maxLag* 阶的自相关系数。
- [`autocorr`](../by-name/a/autocorr.md) — 计算 *X* 的 *lag* 阶自相关系数。请注意，计算时两个时间序列所用的均值均为 *X*的均值，而非两个时间序列各自的均值。
- [`corr`](../by-name/c/corr.md) — 计算 *X* 和 *Y* 之间的相关性（correlation）。
- [`corrMatrix`](../by-name/c/corrMatrix.md) — 设矩阵 *X* 有 n 列，将其每一列作为一维向量，求 n 维向量的相关矩阵（correlation matrix）。
- [`distance`](../by-name/d/distance.md) — 计算 *X* 和 *Y* 之间的距离（单位为米）。
- [`ewmCorr`](../by-name/e/ewmCorr.md) — 返回 *X* 和 *other* 的指数加权移动相关系数。该函数必须指定 *com*, *span*, *halfLife*, *alpha* 四个参数中的一个。
- [`euclidean`](../by-name/e/euclidean.md) — 若 *X* 和 *Y* 是标量或向量，计算 *X* 和 *Y* 之间的欧氏距离。
- [`kendall`](../by-name/k/kendall.md) — 计算 *X* 和 *Y* 的 Kendall 相关性系数。计算时忽略 NULL 值。
- [`mutualInfo`](../by-name/m/mutualInfo.md) — 计算 *X* 和 *Y* 的互信息。
- [`rowEuclidean`](../by-name/r/rowEuclidean.md) — 若 *X* 和 *Y* 同时为向量/矩阵，按行计算 *X* 和 *Y* 之间的欧式距离。若 *X* 和 *Y* 同时为索引矩阵，会对齐标签，对标签相同的行进行计算，标签不同的行直接返回 NULL。

- [`neville`](../by-name/n/neville.md) — 该函数根据 *resampleRule*, *closed*, *origin* 确定的采样规则，对 *X* 进行重采样操作。并根据重采样后的 *X*，对 *Y* 进行 neville 插值（Neville Interpolation）。
- [`pchipInterpolateFit`](../by-name/p/pchipInterpolateFit.md) — 对一组数值向量 *X* 和 *Y* 进行分段三次 Hermite 多项式插值（PCHIP）。
- [`spline`](../by-name/s/spline.md) — 该函数根据 *resampleRule*, *closed*, *origin* 确定的采样规则，对 *X* 进行重采样操作。并根据重采样后的 *X*，对 *Y* 进行三次样条插值（Cubic Spline Interpolation）。
- [`splrep`](../by-name/s/splrep.md) — `splrep` 全称 Spline Representation，即样条插值函数，要求找到一维曲线的 B 样条表示。在给定数据点集 (x[i], y[i]) 后，使用该函数可确定在区间 x[0] <= x <= x[size(x)-1] 上的度数为 3 的平滑样条逼近。注意，若 *x*, *y*, *t* 传入参数中包含 NULL 将直接填充为 0。
- [`splev`](../by-name/s/splev.md) — `splev` 全称 Spline Evaluation，即评估 B 样条曲线或其导数。给定 B 样条表示的节点和系数，使用该函数可求解平滑多项式及其导数的值。若 x, tck 传入参数中包含 None 将直接填充为 0。
