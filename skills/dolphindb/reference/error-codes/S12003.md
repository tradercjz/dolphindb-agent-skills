# S12003

## 错误代码

S12003

## 报错信息

Failed to serialize level file {BlockType}: filepath\_=xxx. RefId: S12003

Failed to deserialize level file {BlockType}: filepath\_=xxx.RefId: S12003

其中 {blockType}是 PKEY 引擎的 LevelFile 的块类型，可以是 header、data block、index block index、data
block index、filter、footer。

## 错误原因

在刷盘过程中写入文件失败时，将会打印此日志并重试。在合并过程中文件写入失败时，将会打印此日志并回滚此次合并。

反序列化过程中失败的可能原因如下：

* 兼容性问题。比如用旧版本 server 读取新版本 server 写入的数据。
* 磁盘或文件系统故障。
* LevelFile 文件损坏。

## 解决办法

1. 检查是否因为磁盘已满或损坏，导致此异常。
2. 检查文件系统是否故障。
