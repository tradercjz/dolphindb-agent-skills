# S03042

## 错误代码

S03042

## 报错信息

The number of result columns must match the number of metrics. RefId: S03042

## 错误原因

创建时序引擎时，`createTimeSeriesEngine` 中 *metric* 参数指定的指标都会输出到
*outputTable* 中，因此必须确保 *outputTable* 中的计算结果列的数量和 *metric*
指定的指标数量一致。否则会出现该报错。

## 解决办法

必须确保 *outputTable* 中的计算结果列的数量和 *metric* 指定的指标数量一致。参考 createTimeSeriesEngine 中
*outputTable* 参数说明，为 *outputTable* 指定正确的 schema。
