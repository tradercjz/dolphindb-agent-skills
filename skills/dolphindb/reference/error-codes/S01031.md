# S01031

## 错误代码

S01031

## 报错信息

[TieredStorageMgmt] : To configure coldVolumes with S3 volumes, the awss3 plugin must
be preloaded by configuring preloadModules. RefId:S01031

## 错误原因

由于分级存储需要使用 AWS S3 插件，所以开启分级存储功能后，server 在初始化时，会检查是否通过配置项 *preloadModules* 提前加载了
AWS S3 插件。如未提前加载插件，则该错误抛出。

## 解决办法

在集群配置中添加 *preloadModules=plugins::awss3*。
